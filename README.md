# Shooting_Ball
자바스크립트로 게임 만들기를 연습 중입니다.
I'm practicing making games with JavaScript.
이 프로젝트는 HTML5 Canvas와 Vanilla JavaScript를 사용해 제작된 로컬 2인용 슈팅 게임입니다.
A local 2-player top-down shooting game built with HTML5 Canvas and Vanilla JavaScript.

🕹 조작 방법 (Controls)
Player 1 (Blue)
이동 (Move): W, A, S, D
발사 (Shoot): F
재장전 (Reload): V

Player 2 (Red)
이동 (Move): I, J, K, L
발사 (Shoot): ; (세미콜론)
재장전 (Reload): P

공통 (Common)
게임 시작 (Start): 아무 키나 누르세요 (Press any key)
재시작 (Restart): 게임 오버 화면에서 R 키 입력
